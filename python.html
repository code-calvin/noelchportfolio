How I built a
Desktop Application

What is a Desktop Application? A desktop application is an application that
runs seperate from your browser. Your browser is a desktop application as it
does not require a browser to execute it. With that explanation, let's begin

To start off, I had a tool which could build the Source Code for the NodeJS Discord Bots. I took that tool, and turned it into a stand alone, graphical application. I kept the same concept, which was:
1. Get input from the user
2. Store that input in variables
3. Take the input, and create source code
4. If the folder exists, it will not create a new one
5. The application must have a loading bar, or an intro (I used loading bar)

With that in mind, I could break this problem even further. To take the input I could use the tkinter module in Python, the Entry() widget. I can do this as so:
var = Entry()

I can turn that Entry widget into a variable by using the get() method. This takes the input inside the entry and I can store it in a variable. aaaaas
